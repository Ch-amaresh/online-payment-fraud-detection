# Online Payment Fraud Detection

This project detects fraudulent online payment transactions using Machine Learning and provides a web interface built with Streamlit.

## Tech Stack

* Python
* Streamlit
* Machine Learning
* Neo4j
* JavaScript Visualization

## Installation

pip install -r requirements.txt

## Run the Application

streamlit run app.py

## Features

* Fraud detection prediction
* Interactive UI
* Graph visualization of transactions
